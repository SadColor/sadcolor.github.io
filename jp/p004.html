Linking nouns with the の particle

{{ imgR }} {{ frog_k }}

The の particle is used for linking two nouns together.
Often times, this is used to describe an ownership relation, like "my drawing" or something.
However, you can also use it to say that someone is something, the way that you might say "my friend Lee" or something like that. This use is frequently used with the phrases like 女おんなの人ひと or 男おとこの人ひと, which you should say instead of just or 女おんな or 男おとこ.
I'm not sure how to word that well, so let's just move on to the examples, yeah?

{{ triR }} {{ reibuntop }}
こちらは隣となりの達哉たつやさんです。 日本語にほんごの宿題しゅくだいはいつも難むずかしい。
This is my neighbor, Tatsuya. Japanese homework is always difficult.
これは何なにかの間違まちがいだろう。 私わたしの猫ねこの目めが黄色きいろい。
This is probably some kind of mistake.
(lit. "mistake of something")		 My cat's eyes are yellow.
微積分ひせきぶんの質問しつもんがある。 友ともだちのつきろは私わたしを裏切うらぎった。
I have a calculus question (to ask). My friend Luniro betrayed me.
{{ cornerR }}

Letting that last example sentence slide, の is a particle that has a lot of uses.
While we won't cover them just yet, just remember that you might see の in a lot of other places that don't have this meaning of connecting two nouns.
This includes the infamous explanatory の, a notoriously difficult usage for Japanese learners. We'll get to learn all about that some other time...

{{ triL }}

The と particle

{{ imgR }} {{ frog_t }}

The と particle is used to list multiple objects, like when we use the English word "and".
In a longer list, after the first と you can use a comma (or pause, in speech) between items instead of saying と every time. It's kind of like the reverse of English where we say "and" at the end of lists instead of the beginning.
An important fact about the と particle is that it is an exhausive listing, meaning that by using the と particle you're implying that there are no other members of the list you're making.
We'll talk about non-exhaustive listing in just a sec, but for now have some examples!

{{ triR }} {{ reibuntop }}
私わたしと友ともだちは一緒いっしょに食たべた。 先生せんせいは私わたしとクラスメイトを叱しかった。
My friend and I ate together. The teacher scolded me and a classmate.
卵たまごと牛乳ぎゅうにゅうとアイスを買かう。 カモとカエル、フクロウが一番いちばん好すき。
I'm gonna buy eggs, milk, and ice cream. I like ducks, frogs, and owls the most.
{{ cornerR }}

Once again, と is a particle that has a few other uses, not just this one.
However, one of them is not connecting clauses of a sentence like the English word "and" can. We'll learn ways to connect multiple clauses later, but for now remember that you shouldn't use と to try to try to link clauses. It can only connect nouns together in a list.

{{ triL }}

Non-exhaustive listing using や and とか and など

{{ imgL }} {{ frog_n }}

In English, if we're giving an incomplete list, we might start it off by saying "including", or say "...and others" or "...et cetera" after we finish the list.
In Japanese, we can convey this concept using the や particle.
It works exactly the same as the と particle we just discussed, including the thing about the commas.
とか is a more casual version of や that works basically the same. However, you can also use とか after the last element of your list if you want (it doesn't change the meaning or anything). And finally, if you want to really emphasize the other objects on the list you're leaving out, you can use など after the last word, which basically means et cetera.
If you use とか or など after the last element of your list, you don't (usually) have to use a particle afterwards. Of course, still using the particle is just fine.

{{ triL }} {{ reibuntop }}
木村きむらや雪子ゆきこは英語英語が話はなせる。 音楽おんがくはロックとかメタルとか（が）好すき。
Kimura and Yukiko (and others) can speak English. I like stuff like metal and rock music.
あの店みせはパンやケーキやクッキーなどを売うる。 蛍光けいこうペンとか付箋紙ふせんしなどは意外いがいと便利べんり 。
That shop sells bread, cakes, cookies, and so on. Highlighters and sticky notes are surprisingly useful.
{{ cornerR }}

Other uses of とか and など

{{ imgR }} {{ frog_t }}

While we're here, I want to note some other uses of とか and など that I think are still pretty related to their normal meanings and are also useful.
You can use both particles after just one word, which still has the same meaning of "stuff like" or "et cetera".
Both can also be used together with the の particle we discussed earlier, which lets you be even more flexible when modifying nouns.
など is also often used to emphasize that one doesn't or shouldn't do something, or anything close to it.
I don't think とか is really used for that, but I'm not sure. In the below sentences you can use either など or とか except for the last one (probably).

{{ triR }} {{ reibuntop }}
ぷよぷよとかのパズルゲームが分わからない。 数十年すうじゅうねん前まえ、携帯けいたいなど存在そんざいしなかった。
I don't understand puzzle games like Puyo Puyo. Decades ago, there weren't things like cell phones.
迷信めいしんとか信しんじない。 あいつは人ひとの気持きもちなど気きにかけない。
I don't believe in superstitions or anything like that.
*using など will sound a little more dismissive of superstitions, I think		 They don't even care about people's feelings.
