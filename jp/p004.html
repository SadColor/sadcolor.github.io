The と particle

{{ imgR }} {{ frog_t }}

The と particle is used to list multiple objects, like when we use the English word "and".
In a longer list, after the first と you can use a comma (or pause, in speech) between items instead of saying と every time. It's kind of like the reverse of English where we say "and" at the end of lists instead of the beginning.
An important fact about the と particle is that it is an exhausive listing, meaning that by using the と particle you're implying that there are no other members of the list you're making.
We'll talk about non-exhaustive listing in just a sec, but for now have some examples!

{{ triR }} {{ reibuntop }}
私わたしと友ともだちは一緒いっしょに食たべた。 先生せんせいは私わたしとクラスメイトを叱しかった。
My friend and I ate together. The teacher scolded me and a classmate.
卵たまごと牛乳ぎゅうにゅうとアイスを買かう。 カモとカエル、フクロウが一番いちばん好すき。
I'm gonna buy eggs, milk, and ice cream. I like ducks, frogs, and owls the most.
{{ cornerR }}

Like a lot of particles, と is a particle that has a few uses.
However, one of them is not connecting clauses of a sentence like the English word "and" can. We'll learn ways to connect multiple clauses later, but for now remember that you shouldn't use と to try to try to link clauses. It can only connect nouns together in a list (in this usage, anyways).

{{ triL }}

Non-exhaustive listing using や and とか and など

{{ imgL }} {{ frog_k }}

In English, if we're giving an incomplete list, we might start it off by saying "including", or say "...and others" or "...et cetera" after we finish the list.
In Japanese, we can convey this concept using the や particle.
It works exactly the same as the と particle we just discussed, including the thing about the commas.
とか is a more casual version of や that works basically the same. However, you can also use とか after the last element of your list if you want (it doesn't change the meaning or anything). If you do use とか last, you don't have to use a particle afterwards.
And finally, if you want to really emphasize the other objects on the list you're leaving out, you can use など after the last word, which basically means et cetera.

{{ triL }} {{ reibuntop }}
木村きむらや雪子ゆきこは英語英語が話はなせる。 音楽おんがくはロックとかメタルとか（が）好すき。
Kimura and Yukiko (and others) can speak English. I like stuff like metal and rock music.
あの店みせはパンやケーキやクッキーなどを売うる。 蛍光ペンとか付箋紙などは意外と便利 。
That shop sells bread, cakes, cookies, and so on. I like ducks, frogs, and owls the most.
{{ cornerR }}

Using adjectives in sentences

{{ imgR }} {{ frog_n }}

Now that that's out of the way, let's discuss how to use adjectives!
Just like English, you can stick adjectives in front of nouns. Just remember that な adjectives will have a な between the adjective and the noun.
And, as talked about in the first lesson, you can also use them as predicates.
い adjectives cannot have だ afterwards, but な adjectives are supposed to (although it is usually dropped). In the polite form, both types of adjectives use です afterwards.

{{ triR }} {{ reibuntop }}
安やすいパソコン 好すきなもの 本当ほんとうの話はなし
a cheap computer a thing I like a true story
このパソコンは安やすい（です）。 こんなものが好すき（です）。 この話はなしは本当ほんとう（です）。
This computer is cheap. I like things like this. This story is true.
{{ cornerR }}

Changing adjectives to adverbs

{{ imgL }} {{ frog_n }}

Next, I want to briefly discuss changing adjectives to adverbs.
Adverbs are used to modify verbs and adjectives, rather than nouns.
For い adjectives, you simply change the い on the end to a く.
For な adjectives, you simply use the に particle after the adjective.
Note that in English, we often use phrases as adverbs in addition to single-word adverbs.

{{ triL }} {{ reibuntop }}
英語えいごがよく分わからない。 VRヘッドセットはすごく高たかい。
I don't understand English well. VR headsets are super expensive.
この謎なぞを簡単かんたんに解く。 この問題もんだいは本当ほんとうに大変。
I will easily solve this riddle. This problem is really tough.
{{ cornerR }}

Wait, what the heck is a よく?
Well, you're probably familiar with the adjective いい, which means "good". This word actually comes from 良よい. You may be able to see that, if we follow the rule for changing an い adjective to an adverb, よく actually comes from 良よい instead of いい.
All inflections of いい besides the base form fall back on the よい reading. This includes when it is in compound words that include いい, like かっこいい, 気持きもちいい, and 仲なかいい.

{{ triR }}

Negative and past tense adjectives

{{ imgL }} {{ frog_t }}

Next, let's talk about the negative form of adjectives!
い adjectives negate by changing the final い into くない.
な adjectives negate by using the negative copula じゃない.
You might notice that the negative ending, ない, also ends in an い. In fact, it inflects just like an い adjective and negative adjectives can be put in front of words just like normal ones!
And, just like a normal い adjective, you cannot use だ afterwards and you add です to be polite.
An alternate polite form you can use is using ありません, the negative polite form of ある, instead of ないです.

{{ triL }}
高たかくないホテル 好すきじゃない食たべもの よくなくない
a hotel that is not expensive a food that I don't like not not good (= not bad = good)
それは確たしかじゃない（です）・確たしかじゃありません。 今日は暑あつくない（です）・暑あつくありません。
That is not certain. Today is not hot.
{{ cornerR }}

On the note of negative い adjectives...
Last section, we just talked about how you can make and adverb by changing the い to a く. If we apply this to a negative い adjective (using 正ただしい as an example), then we get 正ただしく (correct) and ない (to not be, to not exist).
Putting these together, we can see that the meaning of 正ただしくない is "to not be correct". It's actually pretty literal!

{{ triR }}
{{ imgL }} {{ frog_t }}

Up next are past-tense adjectives.
い adjectives indicate past tense by changing い to かった.
な adjectives use the past-tense copula, だった.
Once again, these can go in front of nouns or used in the predicate just like normal.
The polite forms of the two types of adjectives are different. You use です after past-tense い adjectives, and でした instead of だった for な adjectives.
Another note about adjectives is that they express a constant state of being. Therefore, using a past-tense adjective in front of a noun implies that it currently is no longer that adjective.

{{ triL }}
白しろかった壁かべ 嫌きらいだった食たべ物もの お金かねが必要ひつようだった（でした）。
a wall that was white (but is not anymore) a food that I hated (but I do not anymore) I needed money. (lit. money was necessary)
映画えいがは怖こわかった（です）。 昨日きのうは楽たのしかった（です）ね。 前まえの先生せんせいは厳きびしかった（です）。
The film was scary. Yesterday was fun, wasn't it? The teacher before was strict.
{{ cornerR }}

Just like with their negative forms, the past tense of い adjectives isn't completely arbitrary.
In the last lesson, we talked about how the past-tense copula, だった, is a contraction of であった.
In the same way, ～かった is a contraction of ～くあった, where あった is the past tense of ある.
Breaking it down this way, the meaning is very literal, just like with the negative form.

{{ triR }}
{{ imgL }} {{ frog_p }}

As you may expect, you can combine the negative form and the past tense together!
The negative inflection comes first, followed by the past tense. This means that い adjectives will end with くなかった and な adjectives will end with じゃなかった. And, as you may expect, nothing changes about how you can use them.
The polite forms work like past tense: you can add です afterwards, or you can use ありませんでした instead of なかったです.

{{ triL }}
優やさしくなかった人ひと 。 。
a person who was not kind (but is now)
。 。 。
{{ cornerR }}

Using the て form to use multiple adjectives at once

{{ imgR }} {{ frog_n }}

Alright, this is the last thing I want to cover here.
In English, we can just string adjectives together, but in Japanese, when we use a base-form い or な adjective, it indicates a final adjective.
If we want to use multiple adjectives in a row, we'll have to use something called て form, which is used primarily for linking together words and phrases.
For い adjectives, change the final い to くて.
For な adjectives, use the で particle.
Note that て form words are tenseless, so the tense of the entire adjective phrase will be determined by the last adjective that you use.

{{ triR }}
猫ねこは丸まるくてかわいい。 丈じょうはやさしくて素敵すてきで強つよい人ひと。 あの町まちは古ふるくて退屈たいくつだった。
Cats are round and cute. Jou is a kind, wonderful, strong person. That town was old and dull.
宿題しゅくだいは長ながくて難しい。 景色けしきは鮮あざやかで綺麗きれい。 紙かみは高たかくなくて便利べんり。
The homework is long and difficult. The scenery was vivid and beautiful. Paper is useful and not expensive.
{{ cornerR }}
