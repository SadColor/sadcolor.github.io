What constitutes a sentence in Japanese?

{{ imgR }} {{ frog_t }}

In English, you may know that sentences require a subject and a verb (and object for a transitive verb), and that your clause must form a "complete thought" (with some exceptions like answering a question).

{{ triR }} {{ reibuntop }}
I cried. {{ maru }} She is cool. {{ maru }} Going home. {{ batsu }}
{{ cornerR }}
{{ imgL }} {{ frog_n }}

Unlike English, Japanese is a much more context-sensitive language.
Accordingly, you can have totally valid sentences with just one word.

{{ triL }} {{ reibuntop }}
泣ないた。{{ maru }} かっこいい。{{ maru }} 帰かえる。{{ maru }}
(I) cried. (She is) cool. (I'm) going home.
{{ cornerR }}

Okay, but I wanna make sentences that are longer than one word...

{{ imgR }} {{ frog_t }}

Japanese is considered a topic-comment language.
In other words, you have a topic, and then what follows the topic is making a comment about said topic.
The basic way to do this is using 「topicはcomment」, with は being the topic particle.
＊Note that when used as a particle, は is pronounced わ.

{{ triR }} {{ reibuntop }}
猫ねこは寝ねる。 ここは地獄じごく。 ポケモンは面白おもしろい。
Cats sleep. This is hell. Pokémon is interesting.
{{ cornerR }}
{{ imgL }} {{ frog_k }}

Note that using this structure with a verb can be a little tricky.
Some intransitive verbs like 始はじまる, 開あく, or つく refer back to the subject of the verb, and require the particle が.
I'll explain more about the various particles in future lessons, but for now, please check example sentences when you want to use intransitive verbs.

{{ triL }}

You almost never want to use a first-person noun (like 私わたし, for instance) as the topic of your sentence. If you ever say 私わたしは, you usually have some contextual reason like giving your opinion, contrasting yourself with someone else, or something like that.
More on this in a future lesson.

{{ triR }}
{{ reibuntop }}
猫ねこは寝ねる。 {{ maru }} 猫ねこは見みえる。{{ batsu }} 猫ねこが見みえる。{{ maru }}
Cats sleep. A cat is visible. (wrong) A cat is visible.
{{ cornerR }}

A second basic sentence type

{{ imgL }} {{ frog_n }}

Speaking of verbs that require が, two very fundamental verbs are ある and いる.
These verbs are used to say something exists and have close to the same meaning as the English word "be".
You can use them in the structure「thingが(placeに)ある・いる」.
＊いる is used for animate objects while ある is used for everything else.

{{ triL }} {{ reibuntop }}
ベッドがある。 ベッドにいる。
There is a bed. (I'm) in bed.
ベッドが部屋へやにある。 携帯けいたいにゲームがある。
There is a bed in the room. There are games on my phone.
{{ cornerR }}

Because you mark everything with particles in Japanese, the sentence order is pretty flexible.
As you can see, the last two example sentences use the same structure, but the order of thingが and placeに are swapped.

{{ triR }}

Mixing the two

{{ imgL }} {{ frog_n }}

We can actually combine these two types of sentences to say 「topicは(placeに)thingがexists」.
This isn't usually used to say that something exists somewhere physically (that was the previous section).
Instead, by choosing a topic, and commenting on it by saying something exists, this usually means that the topic has something.

{{ triL }} {{ reibuntop }}
ドーナツ屋やは姉あねがいる。 高こうは自転車じてんしゃがある。
Donut Vendor has an older sister. Kou has a bicycle.
ショーンはプログラミングに興味きょうみがある。 トカゲは後うしろに尻尾しっぽがある。
Sean has an interest in programming. Lizards have a tail on their rear.
{{ cornerR }}

Describing something

{{ imgR }} {{ frog_p }}

Because it's a very closely related point, I also want to tell you how to describe characteristics of things. The structure is like this: 「topicはfeatureがadjective」.
Pretty similar, right? Have some examples.

{{ triR }} {{ reibuntop }}
シーポンは耳みみが長ながい。 ルービックは毛けが黒くろい。
Sheepon has long ears. Rubic has black fur.
ジューンは絵えが上手じょうず。 スマホのカメラは画質がしつが高たかい。
June is good at drawing. Smartphone cameras have good image quality.
{{ cornerR }}
